Nous sommes 5 étudiants en deuxième année de Cycle informatique et Réseaux à l'ISEN Brest.

Pendant une semaine, nous avons développé ce site, ce jeu, ainsi qu'un logiciel
permettant d'en configurer certains paramètres avant une partie.

Le but de ce projet est de vous montrer une partie des connaissances que nous avons acquises,
et que vous pourrez acquérir si vous choisissez cette formation.

Technologies utilisées pour ce projet:
Phaser HTML CSS JavaScript jQuery JAVA

L'équipe: Evan Guélard, Joseph Caillet, Yohann Vinez, David Delemotte, Brendan Abolivier





En téléchargeant le projet, vous pouvez ouvrir le logiciel permettant de modifier certains paramètres
du jeu avant de jouer, ou modifier directement ces paramètres dans le fichier variables.js qui se trouve dans le dossier assets.

Même sans connaissances en programmation, vous pouvez ajouter de nouveaux niveaux
avec le logiciel Tiled, ou tout simplement téléchargez le projet.
